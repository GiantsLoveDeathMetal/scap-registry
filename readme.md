
<h1 align="center">
  Scap Registry
</h1>

<h4 align="center">
  Server side application for note storage.
</h4>

## Requirements

```bash
pip install flask
pip install requests
```

## Setup debugger for development

```bash
export FLASK_APP=scap_registry/app.py
export FLASK_DEBUG=1
python -m flask run
```
